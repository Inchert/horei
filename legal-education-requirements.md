## 法令の基礎知識

### 法令の体系

労働安全衛生に関する法令は、以下のような階層構造になっています。

```
労働安全衛生法（法律）
    │
    ├── 労働安全衛生法施行令（政令）
    │       │
    │       └── 労働安全衛生規則（省令）
    │               │
    │               ├── 安全衛生特別教育規程（告示）
    │               │
    │               └── 各種特別規則
    │                   ├── クレーン等安全規則
    │                   ├── ゴンドラ安全規則
    │                   ├── ボイラー及び圧力容器安全規則
    │                   ├── 酸素欠乏症等防止規則
    │                   ├── 有機溶剤中毒予防規則
    │                   ├── 特定化学物質障害予防規則
    │                   ├── 鉛中毒予防規則
    │                   ├── 四アルキル鉛中毒予防規則
    │                   ├── 石綿障害予防規則
    │                   ├── 粉じん障害防止規則
    │                   ├── 高気圧作業安全衛生規則
    │                   ├── 電離放射線障害防止規則
    │                   └── 除染電離放射線障害防止規則

    └── 通達（行政解釈・運用指針）
        ├── 基発（労働基準局長通達）

```

### 教育の省略について

**労働安全衛生規則第40条第3項**により、事業者は、教育事項の全部又は一部について十分な知識及び技能を有していると認められる者については、当該事項に関する教育を省略することができます。

ただし、省略できるのは「十分な知識及び技能を有していると認められる」場合に限られ、単に経験があるというだけでは省略できません。省略する場合は、事業者がその判断根拠を明確にしておく必要があります。

## 職長等教育

### 職長・安全衛生責任者教育（建設業向け）

#### 概要

建設現場などで労働者を指揮する職長は、労働者の健康と安全を確保する重要な立場にあります。事業者は職長などの要職に安全衛生教育（職長教育）を行なうよう労働安全衛生法に規定されています。

建設業では、安全衛生責任者について職長が兼ねることが多いため、厚生労働省では「職長教育」と「安全衛生責任者教育」を統合した「職長・安全衛生責任者教育」の実施を推進しています。

#### 法令根拠一覧

| 根拠種別 | 条文・通達番号 | 内容 |
|---------|-------------|------|
| 法律 | **労働安全衛生法第60条** | 職長等の教育義務 |
| 法律 | **労働安全衛生法第16条** | 安全衛生責任者の選任義務 |
| 法律 | **労働安全衛生法第15条** | 統括安全衛生責任者の選任義務 |
| 政令 | **労働安全衛生法施行令第19条** | 職長教育の対象業種 |
| 省令 | **労働安全衛生規則第40条** | 職長教育のカリキュラム・時間 |
| 省令 | **労働安全衛生規則第40条第3項** | 教育の省略規定 |
| 省令 | **労働安全衛生規則第18条の4** | 安全衛生責任者の職務 |
| 通達 | **令和3年1月25日付通達** | オンライン教育の要件 |

#### 労働安全衛生法第60条（条文全文）

```
（職長等の教育）
第六十条　事業者は、その事業場の業種が政令で定めるものに該当するときは、
新たに職務につくこととなった職長その他の作業中の労働者を直接指導又は
監督する者（作業主任者を除く。）に対し、次の事項について、厚生労働省令で
定めるところにより、安全又は衛生のための教育を行なわなければならない。

一　作業方法の決定及び労働者の配置に関すること。
二　労働者に対する指導又は監督の方法に関すること。
三　前二号に掲げるもののほか、労働災害を防止するため必要な事項で、
　　厚生労働省令で定めるもの
```

#### 労働安全衛生法第16条（条文全文）

```
（安全衛生責任者）
第十六条　第十五条第一項又は第三項の場合において、これらの規定により
統括安全衛生責任者を選任すべき事業者以外の請負人で、当該仕事を自ら
行うものは、安全衛生責任者を選任し、その者に統括安全衛生責任者との
連絡その他の厚生労働省令で定める事項を行わせなければならない。
```

│関すること                                        │        │

２　事業者は、前項の表の上欄に掲げる事項の全部又は一部について

#### 労働安全衛生規則第18条の4（安全衛生責任者の職務）

```
（安全衛生責任者の職務）
第十八条の四　法第十六条第一項の厚生労働省令で定める事項は、
次のとおりとする。

一　統括安全衛生責任者との連絡
二　統括安全衛生責任者から連絡を受けた事項の関係者への連絡
三　統括安全衛生責任者からの連絡に係る事項のうち当該請負人に
　　係るものの実施についての管理
四　当該請負人がその労働者の作業の実施に関し計画を作成する
　　場合における当該計画と特定元方事業者が作成する法第三十条
　　第一項第五号の計画との整合性の確保を図るための統括安全
　　衛生責任者との調整
五　当該請負人の労働者の行う作業及び当該労働者以外の者の
　　行う作業によって生ずる法第十五条第一項の労働災害に係る
　　危険の有無の確認
六　当該請負人がその仕事の一部を他の請負人に請け負わせている
　　場合における当該他の請負人の安全衛生責任者との作業間の
　　連絡及び調整
```

#### 対象業種

**労働安全衛生法施行令第19条**により、建設業が対象業種として定められています。

| 科目 | 内容 | 時間 | 根拠条文 |

**合計**: 14.5時間（2日間に分けて実施）

#### 安全衛生責任者とは

安全衛生責任者は、元方事業者（元請）の労働者と関係請負人（下請）の労働者が同一の場所で作業をする**50人以上の混在作業現場**において、関係請負人側で選任が義務付けられている責任者です。

統括安全衛生責任者との連絡調整、下請負人の労働者への連絡、作業間の調整などを行います。

### 能力向上教育（職長・安全衛生責任者の再教育）

#### 概要

職長・安全衛生責任者教育修了後、概ね5年経過した方を対象とする再教育です。事業者は、厚生労働省通達により、職長等及び安全衛生責任者に従事する労働者に対し、一定期間（概ね5年）ごと、また、機械設備等に大幅な変更があったときにも再教育を実施するよう求められています。

建設業においては特に、安全衛生責任者について職長が兼ねることが多い事から、従事する職長及び安全衛生責任者の能力向上教育の実施を推進しています。

#### 法令根拠一覧

| 根拠種別 | 条文・通達番号 | 内容 |
|---------|-------------|------|
| 法律 | **労働安全衛生法第19条の2** | 能力向上教育の努力義務 |
| 通達 | **平成29年2月20日付基発0220第3号** | 建設業における職長等及び安全衛生責任者の能力向上教育に準じた教育について |
| 指針 | **能力向上教育指針（平成元年5月22日公示第1号）** | 能力向上教育の実施要領 |

#### 労働安全衛生法第19条の2（条文全文）

```
（能力向上教育）
第十九条の二　事業者は、事業場における安全衛生の水準の向上を図る
ため、安全管理者、衛生管理者、安全衛生推進者、衛生推進者その他
労働災害の防止のための業務に従事する者に対し、これらの者が従事
する業務に関する能力の向上を図るための教育、講習等を行い、又は
これらを受ける機会を与えるように努めなければならない。
```

#### 厚生労働省通達（平成29年2月20日付基発0220第3号）

**「建設業における職長等及び安全衛生責任者の能力向上教育に準じた教育について」**

この通達では、以下の点が示されています：

1. **実施時期**: 職長等教育または安全衛生責任者教育を受けてから概ね5年ごと、又は機械設備等に大幅な変更があったとき
2. **対象者**: 職長等として職務を行っている者、安全衛生責任者として職務を行っている者
3. **教育内容**: 基本項目（5時間）＋専門項目（0.5時間以上）

#### カリキュラム詳細

| 科目 | 内容 | 時間 | 根拠 |
|------|------|------|------|
| 職長等及び安全衛生責任者として行うべき労働災害防止に関すること | 最近の建設業における労働災害の動向、労働災害防止対策の重点事項 | 120分 | 基発0220第3号 別添1 |
| 労働者に対する指導又は監督の方法に関すること | 効果的な指導方法、危険予知活動の進め方 | 60分 | 基発0220第3号 別添1 |
| 危険性又は有害性等の調査等に関すること | リスクアセスメントの最新動向、実施事例 | 30分 | 基発0220第3号 別添1 |
| グループ演習 | 事例研究、危険予知訓練、リスクアセスメント演習等 | 130分 | 基発0220第3号 別添1 |

**合計**: 5時間40分

#### 法的性質

能力向上教育は**努力義務**であり、実施しなくても直接の罰則はありませんが、以下の点で重要です：

1. 安全配慮義務の履行として重要視される
2. 労働災害発生時の事業者責任の判断材料となりうる
3. 元請からの要請により実施が求められることが多い

### 製造業向け 職長教育一般

#### 概要

製造業等で労働者を指揮する職長は、労働者の健康と安全を確保する重要な立場にあります。本教育では、労働安全衛生法第60条による職長等に対する安全衛生教育を実施します。

#### 2023年4月の対象業種追加について

**令和4年政令第297号**により、以下の業種が新たに職長教育の対象業種に追加されました（2023年4月1日施行）：

1. **食料品製造業**: 日本標準産業分類の中分類09（食料品製造業）に該当する事業場
2. **新聞業、出版業（印刷部門に限る）**: 編集から印刷まで一貫して行う事業場
3. **製本業、印刷物加工業**: 日本標準産業分類の小分類153に該当する事業場

## 特別教育一覧

### 特別教育の法的根拠

#### 労働安全衛生法第59条第3項（条文）

```
３　事業者は、危険又は有害な業務で、厚生労働省令で定めるものに

## オンライン教育の法的要件

### eラーニングによる特別教育の実施について

#### 根拠通達

**基発0326第1号（令和2年3月26日）「インターネット等を介したeラーニング等により行われる労働安全衛生法に基づく安全衛生教育等の実施について」**

#### 実施要件

| 要件 | 内容 | 確認方法 |
|------|------|---------|
| 教材 | 各科目の教育内容を網羅した動画・資料 | 教育内容の事前確認 |
| 受講管理 | 受講者の視聴状況を確認できるシステム | ログ管理機能 |
| 質問対応 | 受講者からの質問に適切に回答できる体制 | 講師への質問機能 |
| 本人確認 | 受講者の本人確認を行う方法の確保 | 顔認証、受講前確認等 |
| 実技教育 | 原則として対面で実施（一部例外あり） | 対面実施または要件を満たす場合のみオンライン |
| 修了確認 | 理解度テスト等による修了確認 | テスト合格基準 |

### ZOOM等による双方向教育

#### 根拠通達

**令和3年1月25日付通達**

> 「討議方式として実施する教育について、同一時間に参加した受講者の相互のやりとりが可能となるよう双方向性が確保されていること」

#### 実施要件

| 要件 | 内容 |
|------|------|
| 双方向性 | 講師と受講者、受講者間の相互やりとりが可能であること |
| リアルタイム | 同一時間に参加していること |
| 記録 | 受講状況の記録・保存 |
| 本人確認 | カメラオンによる本人確認 |
| 参加確認 | 受講中の離席・不参加の確認 |

## 記録保存の法的要件

### 特別教育の記録

**労働安全衛生規則第38条**

```
（特別教育の記録の保存）
第三十八条　事業者は、特別教育を行つたときは、当該特別教育の
受講者、科目等の記録を作成して、これを三年間保存しておかなければ
ならない。
```

### 記録すべき事項

| 項目 | 内容 |
|------|------|
| 受講者氏名 | 特別教育を受けた労働者の氏名 |
| 教育の科目 | 実施した教育の科目名 |
| 教育の時間 | 各科目の教育時間 |
| 教育の年月日 | 教育を実施した年月日 |
| 講師の氏名・資格 | 教育を実施した講師の氏名・資格等 |

### 保存期間

**3年間**

ただし、じん肺健康診断等の記録など、より長い保存期間が定められているものもあります。

```

## 関係法令条文

### 労働安全衛生法（抜粋）

```
（安全衛生教育）
第五十九条　事業者は、労働者を雇い入れたときは、当該労働者に対し、
厚生労働省令で定めるところにより、その従事する業務に関する安全
又は衛生のための教育を行なわなければならない。

２　前項の規定は、労働者の作業内容を変更したときについて準用する。

３　事業者は、危険又は有害な業務で、厚生労働省令で定めるものに
労働者をつかせるときは、厚生労働省令で定めるところにより、当該
業務に関する安全又は衛生のための特別の教育を行なわなければならない。

（職長等の教育）
第六十条　事業者は、その事業場の業種が政令で定めるものに該当する
ときは、新たに職務につくこととなった職長その他の作業中の労働者を
直接指導又は監督する者（作業主任者を除く。）に対し、次の事項に
ついて、厚生労働省令で定めるところにより、安全又は衛生のための
教育を行なわなければならない。
一　作業方法の決定及び労働者の配置に関すること。
二　労働者に対する指導又は監督の方法に関すること。
三　前二号に掲げるもののほか、労働災害を防止するため必要な事項で、
　　厚生労働省令で定めるもの

（能力向上教育）
第十九条の二　事業者は、事業場における安全衛生の水準の向上を図る
ため、安全管理者、衛生管理者、安全衛生推進者、衛生推進者その他
労働災害の防止のための業務に従事する者に対し、これらの者が従事
する業務に関する能力の向上を図るための教育、講習等を行い、又は
これらを受ける機会を与えるように努めなければならない。
```
